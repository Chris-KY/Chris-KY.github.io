TypeScript Web3 Smart Wallets Gas Abstraction Blockchain Freelance

Overview

  • Worked as a freelance developer on crypto missions for Candide Atelier, a company specializing in smart wallet tooling and account abstraction for Web3.
  • Used TypeScript and the Candide library to develop tools that help developers create secure digital wallets and cover users' gas fees (ERC-4337 account abstraction).
  • Delivered short, focused client missions — each one solving a specific crypto development challenge.

1) Smart Wallets & Account Abstraction

  • Built libraries and tooling around ERC-4337 (Account Abstraction), which allows smart contracts to act as user accounts — enabling gasless transactions and programmable wallet logic.
  • Implemented Paymaster integrations: the mechanism that lets a third party sponsor gas fees on behalf of the user, removing the friction of holding native tokens.
  • Worked with TypeScript throughout — type-safe, maintainable code suitable for open developer tooling.
